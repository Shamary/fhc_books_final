=====================SETUP==================

1) Install dependencies:
    > npm install

2) Setup database(in mysql):
    > source db.sql
    > source procedures.sql 

3) setup environment variables:
    located at /config/environment.js

****start the program****

4) start: 
    > npm start

5) restart: 
    > npm restart

6) stop: 
    > npm stop

7) server logs: 
    > pm2 logs

8) Signin with default credentials:
    username: admin
    password: 1234

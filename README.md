=====================SETUP==================

1) Install dependencies:
    > npm install

2) Setup database(in mysql):
    > source db.sql 

3) setup environment variables:
    located at /config/environment.js

4) start the program
   ***commands***
4a) start: > npm start
4b) restart: > npm restart
4c) stop: > npm stop
4d) server logs: > pm2 logs

5) Signin with default credentials:
    username: admin
    password: 1234

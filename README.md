=====================SETUP==================

1) Install dependencies:
    > npm install

2) Setup database:
    run db.sql in mysql

3) setup environment variables:
    located at /config/environment.js

4) start the program:
    commands:
        start: npm start
        restart: npm restart
        stop: npm stop
        server logs: pm2 logs

5) Signin with default credentials:
    username: admin
    password: 1234
